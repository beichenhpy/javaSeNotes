# 链表

# 1.定义

```java
public class Node {
    //节点内容
    int data;

    //下一个节点
    Node next;
    //构造函数
    public Node(int data) {
        this.data = data;
    }

```

## 2.为节点追加节点

```java
//为节点追加节点
    public Node append(Node node) {
        //找到当前节点
        Node currentNode = this;
        //将下一个节点赋值为当前节点
        while(true){
            //取出下一节点
            Node nextNode = currentNode.next;
            //判断下一节点是否存在
            if (nextNode == null) {
                break;
            }
            //赋给当前节点
            currentNode = nextNode;
        }
        //将要追加的节点追加到当前节点的下一个节点
        currentNode.next= node;
        return this;
    }
```

## 3.插入节点

```java
//插入节点
    public void insertAfter(Node node) {
        //取出下一个节点作为下下一个节点
        Node nextNext = this.next;
        //把新节点作为当前节点的下一节点
        this.next = node;
        //下下一个节点作为新节点的下一个节点
        node.next = nextNext;
    }
```

## 4.下一个节点

```java
public Node Next() {
        return this.next;
    }
```

## 5.得到数值

```java
public int getData() {
        return this.data;
    }
```

## 6.判断是否为最后一个

```java
public boolean isLast() {
        return next == null;
    }
```

## 7.移除节点

```java
public void removeNext(Node node) {
        //取出下下一个节点
        Node newNext = node.next.next;
        //把下下一个节点作为下一个节点
        node.next = newNext;
    }
```

## 8.显示节点所有信息

```java
 //显示所有节点信息
    public void show() {
        Node currentNode = this;
        while(true){
            System.out.print(currentNode.data);
            currentNode = currentNode .next;
            if (currentNode == null) {
                break;
            }
            System.out.print("->");
        }
    }
```
