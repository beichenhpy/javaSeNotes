# 双链表

## 1.定义

```java
public class DoubleNode {
    //上一个节点
    DoubleNode pre = this;
    //下一个节点
    DoubleNode next = this;

    //节点数据
    int data;

    public DoubleNode (int data) {
        this.data= data;
    }
```

## 2.增加节点

```java
 //增加节点
    public void after(DoubleNode node) {
        //找到原来的下一个节点作为下下一个节点
        DoubleNode nextNext = this.next;
        //把新节点作为当前节点的下一个节点
        this.next = node;
        //把当前节点作为新节点的上一个节点
        node.pre = this;
        //把下下一个节点作为新节点的下一个节点
        node.next = nextNext;
        //把新节点作为下下一个节点的上一个节点
        nextNext.pre = node;
    }
```

## 3.测试类

```java
 //双链表

        DoubleNode d1 = new DoubleNode(1);
        DoubleNode d2 = new DoubleNode(2);
        DoubleNode d3 = new DoubleNode(3);

        d1.after(d2);
        d2.after(d3);
        //查看上一个，自己，下一个
        System.out.println(d2.pre.data);//1
        System.out.println(d2.data);//2
        System.out.println(d2.next.data);//3
        System.out.println(d3.next.data);//1
```
